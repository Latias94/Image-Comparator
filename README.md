# Unity Image Comparator

Unity 2021.1.0f1

依赖 Editor Coroutine packages，其他没有 Unity 版本需求。

用 Compute Shader 来查找项目中重复的美术资源，可以调整相似的阈值，自己根据需求来改造插件。

这个是简单的实现，没有后续维护的想法（不想碰 Custom Editor 了 >_<）。
